# Roadmap

## Current Status (v1.0)

✅ **Phase 1: Core Features** (Complete)
- Product search with pricing
- Cart management (view, add, remove, clear)
- Shopping lists support
- Local cart (offline composition, batch sync)
- Needs list (household grocery requests)
- Product preferences (favorites, substitutes, avoid)
- Session management (persistent cookies)

## Planned

### Phase 2: Product Cache & Price Intelligence
- Local product database for fast lookups
- Price history tracking
- Sale detection (price below historical average)
- Smart substitutions based on category + constraints

### Phase 3: Recipe Integration (Mealie)
- Connect to Mealie API for meal planning
- Extract ingredients from recipes
- Map ingredients → Voilà products via preferences
- Generate shopping cart from meal plan

### Phase 4: Learning & Feedback
- Product ratings (quality, value)
- Menu ratings (liked/disliked meals)
- Substitution feedback
- Auto-adjust preferences based on feedback

### Phase 5: Proactive Automation
- Restock reminders based on purchase patterns
- Sale alerts for frequently bought items
- Budget tracking and reports
- Smart menu suggestions (season, sales, history)

## Not Planned

- **Checkout automation**: Requires payment handling, too risky
- **Price scraping at scale**: Against ToS, use sparingly
- **Mobile app**: CLI + Telegram integration is sufficient
